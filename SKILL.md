# Video Title Generator Skill

## 触发词
- "生成标题" / "帮我起标题" / "优化标题"
- "分析视频起标题" / "/titles"

## 核心功能
1. **视频分析**: 不限大小，快速提取视频内容主题
2. **多选标题**: 每个平台生成3个风格不同的标题供选择
3. **学习偏好**: 记录用户的选择，优化未来推荐
4. **参考资料**: 支持导入用户现有标题库作为风格参考

## 平台规则与公式

### 🎵 抖音 (≤55字)

**三种风格:**
| 风格 | 公式 | 示例 |
|------|------|------|
| **A-好奇型** | 悬念+意外结果 | "用了这个方法，效率直接翻10倍？！✨" |
| **B-共鸣型** | 痛点+情绪释放 | "终于有人把这个说清楚了！🔥" |
| **C-实用型** | 数字+即时价值 | "3分钟学会！保姆级教程来了" |

**公式模板:**
- `[人群] + [痛点] + [解决方案]`
- `[数字] + [承诺] + [emoji]`
- `[反常识] + [真相揭示]`

### 📺 B站 (≤80字)

**三种风格:**
| 风格 | 公式 | 示例 |
|------|------|------|
| **A-保姆级** | 【保姆级】+ 关键词列表 | "【保姆级】AI视频剪辑全教程｜零基础｜免费工具｜效率翻倍" |
| **B-测评型** | 【深度测评】+ 结论 | "【深度测评】用了30天AI剪辑软件，优缺点全在这｜值不值得买" |
| **C-干货型** | 【干货】+ 数字+价值 | "【干货】5个改变工作流的AI神器｜亲测有效｜2024必备" |

**关键词分割符:** 用 `｜` (全角) 分割
**类型标签:** 【教程】【测评】【干货】【体验】【攻略】【科普】【避坑】

### 📕 小红书 (≤20字！最严格)

**三种风格:**
| 风格 | 公式 | 示例 |
|------|------|------|
| **A-惊叹型** | 情绪词+核心词 | "救命！这也太好用了吧" |
| **B-种草型** | 必入/闭眼冲 | "AI剪辑神器！闭眼入✨" |
| **C-疑问型** | 反问+意外 | "这个方法真的假的？" |

**10个高转化句式:**
1. `"[引语]"：[犀利观点]`
2. `[亮点前置]+[设置悬念]`
3. `[数字]+[承诺]`
4. `颠覆[刻板印象]`
5. `[人群]必看！`
6. `这个[产品/方法]+[结果]`
7. `别再[错误做法]了！`
8. `[数字]个[类别]，[价值]`
9. `后悔没早点[动作]！`
10. `[意外发现]？！`

### 📱 视频号 (≤30字)

**三种风格:**
| 风格 | 公式 | 示例 |
|------|------|------|
| **A-正式实用** | 直接陈述价值 | "三分钟学会AI视频剪辑" |
| **B-方法论** | 如何/怎样+结果 | "如何用AI提升工作效率" |
| **C-数字型** | 数字+方法/技巧 | "5个实用AI工具推荐" |

**禁忌:** 无emoji、无网络用语、正式语气

### 🌐 YouTube (≤100字)

**三种风格:**
| 风格 | 公式 | 示例 |
|------|------|------|
| **A-SEO优先** | 关键词+年份+受众 | "AI Video Editing | 零基础教程 | 免费工具 | 2024完整版" |
| **B-教程型** | Step by Step+难度 | "AI剪辑入门 | Step by Step | 新手友好 | 中文讲解" |
| **C-对比型** | A vs B + 结论 | "剪映 vs CapCut | 哪个更好用 | 完整对比 | 2024测评" |

**SEO技巧:**
- 关键词放在标题最前面
- 字符数控制在60以内(避免截断)
- 用 `|` 分割关键词
- 包含年份标注(如2024)
- 添加长尾关键词

## 视频分析流程

### 方案1: Gemini快速分析 (推荐，大文件友好)

```bash
# 1. 提取30-60秒音频采样
ffmpeg -i <VIDEO> -ss 00:00:30 -t 60 -q:a 0 -map a /tmp/audio_sample.mp3

# 2. Whisper快速转文字
whisper /tmp/audio_sample.mp3 --model tiny --language zh --output_format txt

# 3. Gemini分析主题并生成标题
gemini "分析以下内容，提取核心主题、关键词、目标受众，然后为5个平台各生成3个风格不同的标题..." -f /tmp/audio_sample.txt
```

### 方案2: 完整字幕分析
```bash
whisper <VIDEO> --model medium --language zh --output_format srt
gemini "基于字幕内容生成多平台标题..." -f <SRT>
```

## 单独重新生成

### 触发词
- "重新生成抖音" / "换一批抖音标题"
- "小红书再来3个" / "B站不满意"
- "YouTube重新生成" / "视频号换一下"

### 支持的平台名
| 输入 | 识别为 |
|------|--------|
| 抖音 / douyin / dy | 抖音 |
| B站 / bilibili / b站 / 哔哩哔哩 | B站 |
| 小红书 / xhs / 红书 | 小红书 |
| 视频号 / 微信视频号 / tencent | 视频号 |
| YouTube / youtube / 油管 / yt | YouTube |

### 使用示例

**场景1: 只重新生成一个平台**
```
用户: 小红书重新生成
AI:
📕 小红书 (新的3个)：
  A. 这工具绝了！ (6字)
  B. 效率神器必入✨ (7字)
  C. 真的好用到哭！ (7字)
```

**场景2: 重新生成多个平台**
```
用户: 抖音和B站重新生成
AI:
🎵 抖音 (新的3个)：
  A. ...
  B. ...
  C. ...

📺 B站 (新的3个)：
  A. ...
  B. ...
  C. ...
```

**场景3: 指定风格重新生成**
```
用户: 小红书再来3个，要更种草风格的
AI: (根据指定风格重新生成)
```

**场景4: 排除已有风格**
```
用户: B站换一批，不要测评型的
AI: (生成3个非测评型标题)
```

### 重新生成规则
1. 保留原有主题分析结果
2. 只刷新指定平台的标题
3. 新标题与之前的不重复
4. 可指定风格偏好或排除某种风格
5. 记录用户的重新生成行为，优化未来推荐
