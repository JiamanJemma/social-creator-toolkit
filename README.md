# Social Creator Toolkit

Claude Code Skill: 全平台内容发布工具包 — 文案生成 + 封面提示词 + 发布策略 + 数据分析

联动 [social-auto-upload](https://github.com/JiamanJemma/social-auto-upload) 形成「创作 → 发布 → 分析」闭环。

## 功能特点

- **全平台文案适配**: 抖音、B站、小红书、视频号、YouTube 一站式生成
- **每平台3个标题选项**: 风格明显区分，供用户选择
- **封面提示词**: 生成 Nano Banana 三种比例（16:9 / 3:4 / 4:3）提示词 + ≤5字封面短标题
- **发布时间推荐**: 基于历史数据 + 平台算法动态推荐，时间错开避免流量清算
- **数据分析反馈**: 追踪各平台发布数据，持续优化标题和时段
- **偏好学习**: 记录选择偏好，越用越懂你的风格
- **爆款公式**: 内置7大心理触发公式

## 支持平台

| 平台 | 标题限制 | 标签规则 | 特殊要求 |
|------|----------|----------|----------|
| 抖音 | ≤30字 | ≤5个 | 必须有简介 |
| B站 | ≤25字 | 10个，顿号分隔，无# | 粉丝动态提醒 ≤233字 |
| 小红书 | ≤20字 | ≤10个 | 避免感叹号用`\|` |
| 视频号 | 短标题≤16字符 | 无限制 | 标题描述可合并 |
| YouTube | ≤100字 | ≤500字符 | SEO优先 |

## 完整工作流

```
字幕分析 → 主题提取 → 标题生成(x3) → 文案适配 → 封面提示词 → 发布时间推荐 → 输出.md
                                                                          ↓
                                                              联动 social-auto-upload 发布
                                                                          ↓
                                                              数据回传 → 优化下次推荐
```

## 安装

```bash
git clone https://github.com/JiamanJemma/social-creator-toolkit.git ~/.claude/skills/social-creator-toolkit
```

## 使用方法

在 Claude Code 中使用触发词：

```
全平台文案              # 生成完整发布方案
帮我起标题              # 只生成标题
生成封面提示词           # 只生成封面提示词
重新生成小红书           # 单独刷新某平台
```

### 场景1: 生成完整发布方案
```
全平台文案 /path/to/subtitle.srt
```

### 场景2: 分析视频生成标题
```
帮我分析这个视频起标题 /path/to/video.mp4
```

### 场景3: 单独重新生成某个平台
```
小红书重新生成
抖音再来3个
B站换一批，不要测评型的
```

## 联动 social-auto-upload

```
social-creator-toolkit          social-auto-upload
━━━━━━━━━━━━━━━━━━━━           ━━━━━━━━━━━━━━━━━━
  内容创作 ─────────────→ 自动发布
    - 标题生成                  - 多平台上传
    - 文案适配                  - 定时发布
    - 封面提示词                - 账号管理
    - 发布时间推荐

  数据分析 ←───────────── 数据回传
    - 播放/互动数据             - 各平台后台数据
    - 标题效果追踪              - 发布状态
    - 时段优化
```

## 7大爆款公式

| 公式 | 核心逻辑 | 示例 |
|------|----------|------|
| 画饼型 | 承诺快速达到理想结果 | "3个月从0到10万粉丝" |
| 损失厌恶型 | 不看会亏/错过损失 | "这5个坑千万别踩" |
| 数字型 | 数字比文字更抓眼球 | "72个免费学习网站" |
| 反认知型 | 颠覆常识引发好奇 | "越努力越穷？真相是..." |
| 提问型 | 引发思考不给答案 | "为什么有人怎么吃都不胖？" |
| 参照物型 | 借熟悉事物做对比 | "比ChatGPT还好用的..." |
| 圈定人群型 | 精准定位目标受众 | "打工人必看" |
