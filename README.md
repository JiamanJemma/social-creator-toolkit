# Video Title Generator for Claude Code

一个 Claude Code Skill，为视频内容自动生成适配多个中国社交平台的标题。

## 功能特点

- **多平台适配**: 抖音、B站、小红书、视频号、YouTube
- **每平台3个选项**: 风格明显区分，供用户选择
- **偏好学习**: 记录用户选择，持续优化推荐
- **视频分析**: 支持任意大小视频，快速提取主题
- **爆款公式**: 内置7大心理触发公式

## 支持平台

| 平台 | 字数限制 | 风格特点 |
|------|----------|----------|
| 抖音 | ≤55字 | 口语化、emoji、制造好奇 |
| B站 | ≤80字 | 【类型】标签、｜分割关键词 |
| 小红书 | ≤20字 | 极简种草风、1个emoji |
| 视频号 | ≤30字 | 正式简洁、无emoji |
| YouTube | ≤100字 | SEO关键词、\|分割 |

## 安装

```bash
# 克隆到 Claude Code skills 目录
git clone https://github.com/YOUR_USERNAME/video-title-generator.git ~/.claude/skills/video-title-generator
```

## 使用方法

在 Claude Code 中使用以下触发词：

```
生成标题
帮我起标题
优化标题
/titles
```

### 场景1: 分析视频生成标题
```
帮我分析这个视频起标题 /path/to/video.mp4
```

### 场景2: 根据主题生成标题
```
帮我起标题，主题是AI视频剪辑教程
```

### 场景3: 优化现有标题
```
优化这个标题："AI工具介绍"
```

### 场景4: 单独重新生成某个平台
```
小红书重新生成
抖音再来3个
B站换一批，不要测评型的
```

## 偏好学习

Skill 会自动记录你的选择偏好到 `preferences.json`：

```json
{
  "style_weights": {
    "douyin": {"A": 5, "B": 2, "C": 3},
    "bilibili": {"A": 3, "B": 1, "C": 6}
  },
  "favorite_words": ["保姆级", "干货", "必看"]
}
```

## 导入参考标题

将你过去的成功标题添加到 `references.txt`，AI会学习你的风格：

```
# 我的抖音爆款标题
这个AI太香了吧！效率直接翻倍✨
终于找到了！免费又好用的剪辑神器
```

## 依赖工具

- **Gemini CLI**: 用于AI分析和生成
- **ffmpeg**: 视频音频提取
- **whisper**: 语音转文字（可选）

## 7大爆款公式

| 公式 | 核心逻辑 | 示例 |
|------|----------|------|
| 画饼型 | 承诺快速达到理想结果 | "3个月从0到10万粉丝" |
| 损失厌恶型 | 不看会亏/错过损失 | "这5个坑千万别踩！" |
| 数字型 | 数字比文字更抓眼球 | "72个免费学习网站" |
| 反认知型 | 颠覆常识引发好奇 | "越努力越穷？真相是..." |
| 提问型 | 引发思考不给答案 | "为什么有人怎么吃都不胖？" |
| 参照物型 | 借熟悉事物做对比 | "比ChatGPT还好用的..." |
| 圈定人群型 | 精准定位目标受众 | "打工人必看！" |
